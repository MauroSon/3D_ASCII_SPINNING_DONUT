# 3D_ASCII_SPINNING_DONUT

The main purpose of this project is implement as the repository title suggests a 3d spinning donut using ASCII characters. This idea is not original, i've seen it in a [video](https://www.youtube.com/watch?v=74FJ8TTMM5E) made by the GreenCode who credited the [original poster](https://www.a1k0n.net/2011/07/20/donut-math.html) and i found it fun enough to try.
